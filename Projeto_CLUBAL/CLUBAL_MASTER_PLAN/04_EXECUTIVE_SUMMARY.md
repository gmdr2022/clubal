# CLUBAL — EXECUTIVE SUMMARY

## O QUE É O CLUBAL

CLUBAL (Club Agenda Live) é uma plataforma leve de comunicação institucional
projetada para funcionar de forma estável em:

- PCs Windows (fracos ou modernos)
- TVs com Windows
- Execução via pendrive
- Ambientes corporativos restritos
- Ambientes totalmente offline

Seu objetivo é exibir informações institucionais de forma clara,
automática e resiliente.

---

## PROBLEMA QUE RESOLVE

Instituições frequentemente precisam exibir:

- Agenda de atividades
- Informações climáticas
- Feriados
- Comunicados importantes
- Avisos urgentes

Soluções tradicionais:

- Dependem de internet constante
- São pesadas
- Exigem navegador aberto
- Travem com facilidade
- Não funcionam bem em TV

CLUBAL resolve isso com:

- Execução local
- Estrutura portátil
- Baixo consumo de recursos
- Compatibilidade total com TV

---

## DIFERENCIAIS TÉCNICOS

- Offline-first
- Fallback automático de paths
- Compatível com modo pendrive
- Não depende obrigatoriamente de APPDATA
- Cache inteligente
- Rotação leve baseada em eventos
- Não recria layout inteiro a cada atualização
- Pillow embutido para compatibilidade de imagem
- Preparado para empacotamento profissional

---

## ESTRUTURA MODULAR

CLUBAL utiliza arquitetura modular baseada em conteúdo.

Módulos atuais:
- Agenda do Dia
- Índices Climáticos
- Calendário de Feriados

Módulos futuros possíveis:
- Avisos institucionais
- Modo emergência
- Mensagens patrocinadas
- Eventos especiais
- QR Code dinâmico

A arquitetura permite expansão sem aumento de complexidade estrutural.
